# Demo patches

Ready-to-bake JSON for the `symbios-audio-cli` binary — three sounds, each in
two forms. Everything here is generated by
[`examples/demo_patches.rs`](../examples/demo_patches.rs); re-run it to
regenerate after a tweak:

```sh
cargo run --example demo_patches
```

There are two parallel **sets** of the same three sounds:

- **Single-graph patches** — one `AudioPatch` DAG baked with the `bake`
  subcommand. There is no sequencer, so rhythm is built inside the graph from
  LFO square waves driving ADSR `gate` ports (each rising edge re-triggers a
  note), including an *inverted* square — a negative LFO `depth` — to land the
  rock backbeat on the off-beat. `bake` does no master limiting, so these were
  auto-gain-staged to peak at 0.9.
- **Sequence recipes** (`seq_*`) — a `SequenceRecipe` baked with the
  `bake-sequence` subcommand. Instruments wire a `Gate` into their envelopes,
  so timed events open real note windows. The mixdown `tanh` soft-clips, so
  levels are governed there.

The single-graph patches are rate- and length-agnostic, so they need an
explicit `--duration` (the CLI default is 1.0 s); the sequence recipes carry
their own `sample_rate`, `bpm`, and `duration_beats`, so `bake-sequence` takes
only input and output paths.

## 1. Triple-caw of a crow

```sh
cargo run --release --bin symbios-audio-cli -- bake --duration 1.6 \
    patches/crow_triple_caw.json crow.wav
cargo run --release --bin symbios-audio-cli -- bake-sequence \
    patches/seq_crow_triple_caw.json seq_crow.wav
```

A bright sawtooth honk through a bandpass formant, a noise rasp, an 80 Hz
"buzz" tremolo, and a per-caw downward pitch glide. Three caws: the
single-graph version times a 1.8 Hz trigger against a 1.6 s bake; the sequence
version schedules three events at beats 0, 1, 2.

## 2. Short rock groove — drums, bass, guitar

```sh
cargo run --release --bin symbios-audio-cli -- bake --duration 8.0 \
    patches/rock_groove.json rock.wav
cargo run --release --bin symbios-audio-cli -- bake-sequence \
    patches/seq_rock_groove.json seq_rock.wav
```

120 BPM, 4/4: synth kick (pitch-blip sine) on 1 & 3, snare (noise + tonal body)
on 2 & 4, hi-hats and bass on straight 8ths, and a root+fifth power-chord
guitar with chorus on the quarter note. Both versions walk an E–G–A
power-chord vamp — the single-graph one with a slow square LFO adding Hz to the
oscillators, the sequence one with `TimePreserving` pitched events.

## 3. Moody cyberpunk-noir ambient soundscape

```sh
cargo run --release --bin symbios-audio-cli -- bake --duration 14.0 \
    patches/cyberpunk_noir_ambient.json ambient.wav
cargo run --release --bin symbios-audio-cli -- bake-sequence \
    patches/seq_cyberpunk_noir_ambient.json seq_ambient.wav
```

A dark drone (detuned saws + sub) under a tritone pad with chorus shimmer,
swept pink/brown noise for rain-and-rumble air, and a sparse two-partial sine
"ping" (a distant neon signal) — all under a big reverb with slow LFOs
breathing the filter cutoffs. The sequence version is a seamless 16 s loop
(`loop_start_beats` + crossfade) with the ping pitched up the scale.
